# Contributing to HSIL

Thank you for your interest in contributing to the Human Semantic Intention Language.

HSIL is an open standard designed to serve as a universal representation of human intention for AI systems, agents, robotics, and synthetic environments.

---

## How to Propose Changes

All changes must be submitted through the HSIP process:

1. Fork this repository  
2. Create a file in `/proposals/` using the naming format:
HSIP-###-short-title.md
3. Use the official template:  
/proposals/HSIP-TEMPLATE.md
4. Submit a Pull Request  

The HSIL Steering Committee (HSC) reviews all proposals.

---

## What You Can Contribute

- Improvements to the spec  
- Parsers / SDKs in any language  
- Examples  
- Research-based insights  
- Documentation  
- Tooling
